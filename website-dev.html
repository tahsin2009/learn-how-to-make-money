Article 2: How the Web Works ‚Äì Understanding HTTP, DNS, Browsers, and Hosting

Word Count: ~1200

💻 What Happens When You Type a URL?

When you enter a website address like www.example.com into your browser, a lot happens behind the scenes to show you that webpage. Understanding this process is crucial to becoming a skilled web developer.

Step 1: DNS Lookup 🔍

DNS stands for Domain Name System. It works like a phonebook for the internet.

  • Your browser asks the DNS server to translate the human-friendly domain (example.com) into an IP address (like 192.0.2.1), which computers use to find each other.
  • This process is called a DNS lookup.

Step 2: Sending an HTTP Request 📥

Once your browser knows the IP address, it sends an HTTP (Hypertext Transfer Protocol) request to the web server hosting the site.

  • This request asks the server to send back the website‚Äôs files.
  • There are different types of HTTP requests, but the most common is GET, which requests data.

Step 3: Server Responds with HTTP Response 📱

The server processes your request and sends an HTTP response.

  • This includes the requested HTML, CSS, JavaScript files, images, etc.
  • The server also sends a status code. For example, 200 OK means everything went well, while 404 Not Found means the page doesn‚Äôt exist.

Step 4: Browser Renders the Page 🌐

Your browser reads the HTML, applies CSS for styling, and runs JavaScript to make the page interactive.

  • The page you see is built live in your browser‚Äôs rendering engine.
  • Modern browsers like Chrome use engines like Blink to perform this.

🌐 What Is Hosting and Why Do You Need It?

Web hosting is where your website ‚Äúlives.‚Äù It‚Äôs a server (a powerful computer) connected to the internet that stores your website files and makes them accessible to users worldwide.

Types of Hosting

  • Shared Hosting: Your website shares server resources with others. It‚Äôs affordable but slower. Examples: Bluehost, HostGator.
  • VPS Hosting: Virtual Private Server gives you more control and power. You get dedicated resources.
  • Dedicated Hosting: You rent an entire server. Used by large websites needing maximum control.
  • Cloud Hosting: Your site is hosted across multiple servers. Scalable and reliable. Examples: AWS, Google Cloud, DigitalOcean.

📚 Domain Names ‚Äì Your Website‚Äôs Address

A domain name is your website‚Äôs address on the internet, like yourwebsite.com.

  • You register domain names through registrars such as GoDaddy, Namecheap, or Google Domains.
  • Domains must be renewed yearly.
  • Choosing a good domain name affects branding and SEO.

⚙️ What Are Web Browsers and How Do They Work?

Web browsers are software applications that fetch, interpret, and display web pages for users. Popular browsers include Google Chrome, Firefox, Safari, and Edge.

Key Browser Components

  • UI: Address bar, back button, bookmarks.
  • Rendering Engine: Translates HTML, CSS, JS into what you see. Example: Blink in Chrome.
  • JavaScript Engine: Runs JavaScript code. Example: V8 in Chrome.
  • Networking: Handles HTTP requests and responses.

🛠️ Hosting Your First Website ‚Äì Basic Overview

To get your website online, you need:

  1. A domain name
  2. A hosting service
  3. Your website files (HTML, CSS, JS, images)

Most hosting providers offer a control panel to upload your files easily, or you can use FTP (File Transfer Protocol) clients like FileZilla.

🌐 What Is HTML?

HTML (Hypertext Markup Language) is the backbone of every webpage. It defines the structure and content of a webpage using elements called tags.

Without HTML, your browser wouldn‚Äôt know how to display text, images, links, or other content.

🔧 Basic Structure of an HTML5 Document

Here‚Äôs the minimal template for any HTML5 webpage:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Explanation:

  • <!DOCTYPE html> declares this is an HTML5 document.
  • <html lang="en"> is the root element with language set to English.
  • <head> contains meta info like encoding and title.
  • <body> contains visible content.

📝 Important HTML Tags to Know

1. Headings 📌

HTML provides six levels of headings: <h1> to <h6>.

<h1>Biggest Heading</h1>
<h2>Second Biggest</h2>

2. Paragraphs 📝

<p>This is a paragraph.</p>

3. Links 🔗

<a href="https://example.com">Visit Example</a>

4. Images 📷

<img src="image.jpg" alt="Description" />

5. Lists 📖

Ordered list:

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Unordered list:

<ul>
  <li>Item one</li>
  <li>Item two</li>
</ul>

🛠️ Creating Your First HTML File

  1. Open Visual Studio Code or your favorite editor.
  2. Create a new file and save it as index.html.
  3. Copy the basic HTML structure above into the file.
  4. Modify the title and content inside the <body> tags.
  5. Open the file in your browser to see your webpage.

💻 Understanding Tags vs Elements vs Attributes

  • Tag: The markup itself like <p> or <a>.
  • Element: The entire item including start tag, content, and end tag. Example: <p>Hello</p>.
  • Attributes: Extra info inside tags to customize elements. Example: href in <a href="url">.

Article 4: Styling Your Webpage with CSS ‚Äì Making Your Site Look Amazing

🎨 What Is CSS?

CSS (Cascading Style Sheets) controls the appearance and layout of your webpages. While HTML builds the structure, CSS makes your site beautiful, colorful, and user-friendly.

🖌 How CSS Works

CSS works by targeting HTML elements and applying styles like colors, fonts, spacing, and positioning.

Example: To make all paragraphs <p> text red, you write:

p {
  color: red;
}

📊 Ways to Add CSS

  1. Inline CSS: Add styles directly to an HTML tag using the style attribute.
    <p style="color: blue;">Text</p>
  2. Internal CSS: Add styles inside a <style> block within the HTML <head>.
    <style> p { color: green; } </style>
  3. External CSS: Create a separate .css file and link it in HTML. This is best for bigger projects.
    <link rel="stylesheet" href="styles.css">

🛠️ CSS Syntax Basics

CSS rules have three parts:

selector {
  property: value;
}
  • Selector: The HTML element you want to style (e.g., p, h1, .class).
  • Property: The style attribute (e.g., color, font-size).
  • Value: The setting for that property (e.g., red, 20px).

🖋 Common CSS Properties

  • color: Text color
  • background-color: Background color
  • font-size: Size of text
  • margin: Space outside elements
  • padding: Space inside elements
  • border: Borders around elements
  • width & height: Size of elements

🌟 Colors in CSS

You can use colors by name, HEX code, RGB, or HSL:

  • Name: red, blue
  • HEX: #ff0000 (red), #0000ff (blue)
  • RGB: rgb(255, 0, 0)
  • HSL: hsl(0, 100%, 50%)

🛠️ Classes and IDs

To style specific elements, you can assign classes or IDs in HTML and target them in CSS.

<p class="highlight">Important text</p>

CSS:

.highlight {
  background-color: yellow;
}

ID example:

<h1 id="main-title">Welcome</h1>
#main-title {
  font-size: 36px;
}

🛠️ Responsive Design Basics

Responsive design makes your website look good on all devices ‚Äî phones, tablets, and desktops.

The meta viewport tag (which you added in Article 3) helps browsers scale your site:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS Media Queries let you apply styles based on screen size:

@media (max-width: 600px) {
  body {
    background-color: lightgrey;
  }
}

💻 What Is JavaScript?

JavaScript (JS) is the programming language that brings interactivity and dynamic behavior to websites. While HTML structures and CSS styles your pages, JavaScript lets you respond to user actions, manipulate content, and create animations.

🛠️ How JavaScript Works in the Browser

JavaScript runs inside the browser‚Äôs JavaScript engine (like V8 in Chrome). It can read and modify HTML elements, handle events like clicks, and communicate with servers.

💻 Adding JavaScript to Your Page

There are three ways to add JavaScript:

  1. Inline: Directly inside an HTML element (not recommended for most cases).
    <button onclick="alert('Hello!')">Click Me</button>
  2. Internal: Inside a <script> tag in your HTML file.
    <script>alert('Hi!');</script>
  3. External: In a separate .js file linked with <script src="script.js"></script>. Best for organization.

📝 JavaScript Syntax Basics

Here are some key concepts and examples:

Variables 📝

Variables store data. Use let or const to declare them.

let name = 'Tahsin';
const pi = 3.14;

Functions 🛠️

Functions are reusable blocks of code.

function greet() {
  alert('Hello!');
}

greet(); // calls the function

Events 👉

JavaScript reacts to user actions like clicks.

document.getElementById('btn').addEventListener('click', function() {
  alert('Button clicked!');
});

Manipulating HTML Elements 📄

You can change content or styles dynamically.

document.getElementById('title').textContent = 'New Title';
document.getElementById('title').style.color = 'blue';

Article 6: Advanced CSS Techniques & Layouts ‚Äì Making Your Website Professional

⚙️ CSS Box Model Explained

The CSS box model is a fundamental concept that determines how elements are sized and spaced on a webpage.

Every HTML element is a rectangular box made of:

  • Content: The actual text or images inside the element.
  • Padding: Space between the content and the border.
  • Border: A line surrounding the padding and content.
  • Margin: Space outside the border, separating the element from others.

Example CSS:

div {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

🛠️ CSS Positioning

Positioning controls how elements are placed on the page.

  • Static: Default position, flows with the document.
  • Relative: Moves relative to its normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Stays fixed relative to the viewport (like sticky headers).
  • Sticky: Toggles between relative and fixed depending on scroll.

🏠 CSS Flexbox ‚Äì Easy Layouts

Flexbox helps create responsive layouts by aligning and distributing space among items in a container.

Basic example:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Flexbox properties include:

  • flex-direction: row or column
  • justify-content: alignment along main axis
  • align-items: alignment along cross axis
  • flex-wrap: whether items wrap onto multiple lines

📄 CSS Grid ‚Äì Powerful Two-Dimensional Layouts

CSS Grid is a more advanced layout system, allowing control over rows and columns.

Example:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

This creates a grid with 3 equal columns and 10px gaps.

🛠️ Responsive Design with Media Queries

Use media queries to change layouts on different screen sizes:

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

🛠️ Transitions and Animations

Make your site more engaging with smooth animations.

Simple transition example:

button {
  background-color: blue;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: green;
}

Article 7: JavaScript DOM Manipulation ‚Äì Making Your Webpages Interactive

💻 What Is the DOM?

The DOM (Document Object Model) is a structured representation of your webpage, like a tree made up of elements (HTML tags). JavaScript can interact with the DOM to change content, styles, and structure dynamically.

🛠️ Accessing DOM Elements

You can select elements in various ways:

  • document.getElementById('id') ‚Äì selects element by its ID.
  • document.getElementsByClassName('class') ‚Äì selects elements by class (returns a collection).
  • document.querySelector('selector') ‚Äì selects the first element matching a CSS selector.
  • document.querySelectorAll('selector') ‚Äì selects all elements matching a CSS selector.

📝 Changing Content and Attributes

Once selected, you can change an element's content or attributes:

const title = document.getElementById('title');
title.textContent = 'New Title';

const image = document.querySelector('img');
image.src = 'new-image.jpg';
image.alt = 'Updated description';

🛠️ Changing Styles

You can also modify styles directly via JavaScript:

const box = document.querySelector('.box');
box.style.backgroundColor = 'lightblue';
box.style.border = '2px solid black';

👉 Adding and Removing Elements

You can create new elements and add them to the DOM, or remove existing ones.

// Create a new paragraph
const p = document.createElement('p');
p.textContent = 'This is a new paragraph.';

// Add it to a container
const container = document.querySelector('#container');
container.appendChild(p);

// Remove an element
const oldItem = document.querySelector('.old-item');
oldItem.remove();

🛠️ Event Listeners

To make your webpage interactive, listen for user actions:

const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  alert('Button clicked!');
});

🚀 Practical Example

Here‚Äôs a simple webpage that changes text and color when you click a button:

<!DOCTYPE html>
<html lang="en">

  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>DOM Manipulation Example</title>
</head>

  <button id="changeBtn">Change Text</button>

  <script>
    const btn = document.getElementById('changeBtn');

      const title = document.getElementById('title');
      title.textContent = 'Text Changed!';
      title.style.color = 'red';
    });
  </script>

</body>
</html>

  height: auto;

    display: none;
  }

    width: 100%;
  }
